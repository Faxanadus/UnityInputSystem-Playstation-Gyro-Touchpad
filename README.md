# UnityInputSystem-Playstation-Gyro-Touchpad
Scripts that make Gyro, Accelerometer, and the Touchpad work with PS4 and PS5 controllers in Unity's InputSystem.

These scripts have been minimally modified from their original versions included with the InputSystem (1.8.2) package to expose the controller's gyro, accelerometers, and the touchpad values to DualShockGamepad.current

A sample project, tested in Unity 2022.3.22f1 with everything necessary to get this to work is provided in Releases -->
